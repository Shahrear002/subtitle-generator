## [1.0.0] - 2026-01-14

### Added
- Initial release of Subtitle Generator
- GUI application built with Tkinter
- AI-powered subtitle generation using OpenAI Whisper
- Support for multiple video formats (MP4, AVI, MKV, MOV, etc.)
- Automatic subtitle file (.srt) generation
- FFmpeg integration for audio extraction
- Windows .exe installer
- Multi-language transcription support (99+ languages)
- Local processing for privacy

### Features
- Simple, intuitive user interface
- File browser for video selection
- Progress indication during subtitle generation
- Automatic subtitle saving to video location
- High-accuracy transcription
